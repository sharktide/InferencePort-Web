What you can do

Run local models

Start and interact with models hosted on your machine (Ollama, local runtimes). Stream responses to the UI and save sessions.

Browse Spaces

Discover, add and preview Hugging Face Spaces and community demos directly from the app.

Share and export

Share website cards and export session data for reproducibility and collaboration.

Privacy-first

All model runs happen locally (unless you choose otherwise). We aim for minimal external calls and transparent behavior.

Get started

1. Download the desktop app from the Download page.

2. Open the Playground to test models without installing anything locally.

3. Visit the Settings to configure model backends and user preferences.
